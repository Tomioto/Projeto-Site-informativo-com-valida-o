O que é a linguagem HTML ?

HTML é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

HyperText Markup Language
O que é a linguagem CSS ?

CSS é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.

Linguagem Cascading Style Sheet
O que é a linguagem BootStrap ?

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.

Exemplo de códigos HTML

1. Tag <body> Sua finalidade é definir o corpo do documento, ou seja, tudo que estiver dentro da tag body será mostrado de alguma forma no conteúdo da página.

2. Tag <head> Providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

3. Tag <div> É separar os conteúdos do código HTML em divisões de espaço/conjunto, sem qualquer peso semântico.

4. Tag <a> Cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

5. Tag <strong> É utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas).

6. Tag <br> Para adicionar quebras de linha em um texto HTML, indicado apenas para casos em que queremos adicionar uma quebra específica em um texto.

Exemplo de códigos CSS

1. Tag <font> É utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

2. Tag <p> Essa tag é usada para definir parágrafos de texto em um site. Você pode definir a fonte, cor e espaçamento entre parágrafos usando propriedades CSS.

3. Tag <style> É utilizado para transferir informações de formatação para tags, para conteúdos, dentro das tags de uma página HTML.

4. Tag <ul> é um elemento HTML que representa uma lista não ordenada. Este elemento tem como característica criar listas não ordenadas no HTML.

5. Tag <span> é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id).

6. Tag <img> É utilizado para inserir imagens em uma página HTML. A imagem adicionada pode ser de diferentes tamanhos, origens e formatos, de que falaremos mais adiante.

Exemplo de códigos BootStrap

1. Tag <button> É utilizada para inserir interatividade em uma página ou formulário e tem como função executar uma determinada ação ao receber um clique do usuário.

2. Tag <section> É utilizada para marcar as seções de conteúdo de uma página. Com Esse elemento agrupamos de forma lógica nosso conteúdo, separando a informação em áreas diferentes.

3. Tag <code> são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

4. Tag <samp> É utilizado para destacar possíveis resultados de algum processo "on line" ou a explicação em algum tutorial em formato pdf, por exemplo.

5. Tag <var> É utilizado para indicar um conjunto de fatores de uma variável, um parâmetro de um sistema.

6. Tag <h1> É um importante elemento que o Google utiliza para determinar o principal assunto abordado em uma página, visto que o título de uma página conceitualmente define seu conteúdo.
